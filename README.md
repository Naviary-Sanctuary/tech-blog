# Naviary Sanctuary Tech Blog

Naviary는 앵무새 전문 생태계를 구축하는 브리딩센터입니다.  
브리딩을 감(感)이 아닌 데이터와 구조로 설계하는 시스템을 지향합니다.
개체 관리, 페어링 전략, 번식 효율, 혈통·건강 이력 관리까지 체계화하는 브리딩 시스템을 운영합니다.

## Development

```sh
bun install
bun dev
```

## Build

```sh
bun build
bun preview
```
